About the Project

Clock Wall is a screen saver that uses a grid of analog clocks to create a digital clock display. When the time changes the it will play an animation to move the hands to their new locations. Any clocks not used in the digital display display the actual time as an analog clock would. It has 12 hour and 24 hour display formats and all of the colors are customizable. It was written in C#.

Installation

To install simply download ClockWall.scr, right click the file and select "Install". A window will then pop up allowing you to configure the screen saver. To change time formats or colors click "Settings...". To do a test run click "Preview". Hit "Apply" when you are done and the scren saver is installed and ready to go!
